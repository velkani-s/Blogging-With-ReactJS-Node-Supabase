# Blog Frontend

This is the frontend for the full-stack blog website built with React, Vite, and Tailwind CSS.

## Features

- Home page with featured content
- Blog list with search functionality
- Individual blog post pages
- Product list and product pages
- Admin panel (placeholder)
- Responsive design
- SEO with dynamic meta tags
- Affiliate disclosure

## Installation

1. Navigate to the frontend directory
2. Run `npm install`
3. Run `npm run dev` to start the development server

## Build

Run `npm run build` to build for production.

## Technologies Used

- React
- Vite
- React Router
- Axios
- Tailwind CSS
- React Helmet Async
